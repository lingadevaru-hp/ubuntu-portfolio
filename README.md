# Lingadevaru Linux Portfolio

Ubuntu-style personal portfolio built with Next.js and Tailwind CSS.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` (or your configured port).

## Build for production

```bash
npm run build
npm run start
```
